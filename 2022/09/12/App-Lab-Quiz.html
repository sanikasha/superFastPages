App Lab Quiz

Sep 12, 2022 • 3 min read

Plan:

Quiz about Madagascar the Movie, that eventually turns into a Computer Science Quiz.

  1. Begins on the Start page with a button to lead to the next page. Music plays in the background.

  2. Another beginning page (with reference to the movie)

  3. First question “Who does Mort worship?”, give 4 options

  4. Each option is clickable, and will lead to “correct” or “incorrect” answer screen

  5. Second question “What animal tries to eat Mort?”, give 4 options

  6. Each option is clickable, and will lead to “correct” or “incorrect” answer screen

  7. Third question “Mort was the first lemur the main characters encountered. True/False”

  8. Each option is clickable, and will lead to “correct” or “incorrect” answer screen

  9. Quiz is hacked! Computer science quiz begins.

  10. Fourth question “Dictionaries: a way of grouping data into key-value relationships. True/False”

  11. Each option is clickable, and will lead to “correct” or “incorrect” answer screen

  12. Fifth question “What is the most productive percentage that off-topic conversation should be in APCSP?”, give 4 options

  13. Each option is clickable, and will lead to “correct” or “incorrect” answer screen

  14. Finish screen, with score

Goals:

  • Create a quiz that starts out Madagascar themed, then changes into a Computer Science quiz.

  • Become more comfortable to Applab

  • Introduction to javascript language

Successes:

  • Were able to create personalized screens

  • Comfortable with onEvent, console.log, and setScreen

  • Were able to manually direct quiz to “correct” or “incorrect” page, when user clicked answer

Growth Opportunities:

  • Was not able to make if/then functions work, and therefore had to write many lines of repeating code

  • Learnings:

  • setScreen(“x”) directs the computer to the next screen, while console.log carries out the action

  • Seeing blocks makes it easier to visual the code when there are so many lines, however seeing text allows you to more
